# 🧠 Socratic AI — RICE Workshop

## 🎯 5 Core Features (PRD Day 17)

1. **Socratic Questioning Engine**  
   → AI hỏi ngược để dẫn dắt tư duy

2. **Adaptive Depth Mode**  
   → Beginner → Advanced (điều chỉnh độ sâu câu hỏi)

3. **Conversation Memory (Context-aware thinking)**  
   → Giữ context để suy nghĩ liên tục

4. **Use-case Templates**  
   → Problem solving / Learning / Decision-making

5. **Insight Summary**  
   → Tóm tắt lại quá trình tư duy

## 🧩 Key Insights

- **Core value:** Socratic Questioning Engine (highest impact)
- **Growth lever:** Use-case Templates (easy adoption)
- **Long-term moat:** Memory (but high effort)

---

## 🧭 2x2 Value vs Effort

### 🚀 Quick Win
**Use-case Templates**
- Low effort
- High reach
- Easy to demo & viral

---

### 🧠 Strategic Bet
**Socratic Questioning Engine**
- Core differentiation
- Defines product identity

---

### ❌ Non-starter (Delay)
**Conversation Memory (full-scale)**
- High effort (infra + privacy)
- Not needed early stage
- Replace with session-based memory

---

### ⚖️ Build Next
- Insight Summary → tăng retention  
- Adaptive Depth Mode → refine UX
